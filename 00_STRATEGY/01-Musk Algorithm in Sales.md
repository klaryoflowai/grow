# Musk Algorithm in Sales

## 1. Simplify

- Reduce the sales process to a few stages with clear exit criteria.
- Use one source of truth for Dream 100, pipeline, scripts, and scorecards.
- Force every meeting note to end with a next step and date.

## 2. Eliminate

- Remove low-volume opportunities that consume management time.
- Remove custom follow-up behavior and replace it with a standard cadence.
- Remove repeated explanation work by documenting scripts and objections once.

## 3. Automate

- Use Dataview to surface stale accounts and signed contracts.
- Use templates so new notes start with correct metadata.
- Later, automate Dream 100 import from CSV into individual notes.

## Practical Rule

If a process cannot be explained in one page or one Canvas lane, it is still too complex.
